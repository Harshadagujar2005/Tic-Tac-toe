Tic Tac Toe

A simple Tic Tac Toe game implemented in Python. This repository contains a Python program that allows two players (or player vs AI, if implemented) to play the classic 3x3 grid game.

The Tic Tac Toe project is a console-based (or GUI, depending on your version) game where:

The first player to align three symbols horizontally, vertically, or diagonally wins.

If all nine squares are filled and no one wins, the game ends in a draw.
